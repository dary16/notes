- 函数内部
		
	- 函数直接执行
		- 非严格模式下
			- 默认指向全局对象
		- 严格模式（前面省略window,浏览器严格模式支持）
			- undefined
			 
		
	- call,apply和bind
		- 把this的值从一个执行环境传入另一个执行环境
			- call和apply的参数差异
				- 函数.call会直接调用函数，并且call的第一个参数就是函数执行时的this，后面的是参数
			- 如果传入的不是对象会调用相对的构造函数，进行隐式转换
		
	- bind
		- 返回一个新函数，内部this指向被改变
		- 只会改变一次
		- bind改变this指向但并不执行函数，返回一个新函数

----------
- 箭头函数内部
	- 与定义时的环境中this一致
		- 点击元素改变
	- bind,call不能修改其内部指向
	- 全局中定义箭头函数
	- 对象内函数定义箭头函数
	- 对象内函数返回箭头函数

- 对象方法内部
	- 当函数作为对象里的方法被调用时
		- 调用该函数的对象
	- 对象后续附属上的方法调用
		- 调用该函数的对象
	- 作为函数内嵌套多层的方法调用
		- 就近绑定

- 原型链上的函数
	- 调用该函数的对象

```javascript
var p = {
	a:1,
	fn:function(){
		console.log(this.a); 1
	}
}
var c = Object.create(p);
console.log(c.a);  1
```
